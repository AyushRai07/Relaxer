# 🌬️ Relaxer App

A simple breathing animation project to help you relax and reduce stress.


## ✨ Features

- Animated breathing cycle (Inhale, Hold, Exhale)
- Looped timing with visual feedback
- Clean UI with relaxing background
- Mobile responsive (✅)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
